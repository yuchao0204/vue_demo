# cesium-vue

##  项目安装(Project setup)
```
npm install / cnpm install 
```

### 编译并热部署的开发环境 (Compiles and hot-reloads for development)
```
npm run serve / cnpm run serve
```

### 编译并压缩的生产环境 (Compiles and minifies for production)
```
npm run build / cnpm run build
```
